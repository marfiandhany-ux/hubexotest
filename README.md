# Playwright Java Automation

This project contains automated test cases written in **Java + Playwright**.  
It is designed for testing [SauceDemo](https://www.saucedemo.com/) web application.

---

## Prerequisites

- [Java 11+](https://adoptium.net/) installed
- [Maven](https://maven.apache.org/download.cgi) installed
- Internet connection (to access the test website)

---

## How to Run Tests

Clone the repository and run the following command:

mvn test
